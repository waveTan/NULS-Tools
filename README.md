# NULS-Tools

## Project setup
```
npm install or yarn
```

### Compiles and hot-reloads for development
```
npm run serve-prod
```

### Compiles and minifies for production
```
npm run build-prod
```

### Run your electron
```
npm run electron:serve
```

### Run your electron:build
```
npm run electron:build
```
